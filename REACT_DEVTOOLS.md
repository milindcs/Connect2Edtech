# React DevTools (Development Helper)

Install the **React DevTools** browser extension for a better development experience while working on this project.

## Recommended link
https://react.dev/link/react-devtools

## Chrome / Edge / Brave / Opera
1. Open the link above.
2. Choose your browser (or click the Chrome Web Store / Microsoft Edge Add-ons option).
3. Click **Add to Chrome/Edge**.
4. Pin the extension if desired.

## Firefox
1. Open the link above.
2. Select the Firefox add-on option.
3. Click **Add to Firefox**.
4. Enable the extension if prompted.

## Verify it’s working
1. Start your dev server.
2. Open the app in the browser.
3. Open DevTools.
4. You should see the **React** tab (or a React badge/section) in DevTools.

> Note: This project already runs in development mode during local dev; React DevTools will help inspect component hierarchy, props, state, and hooks.
